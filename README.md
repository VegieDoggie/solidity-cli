# solidity-framework

en: A General Solidity Development and Testing Framework/CLI.
zh: 通用的开发和测试`Solidity`的快速脚手架

usages:
```cmd
npm i -g solidity-framework

mkdir test1
cd test1
sol init
# or just `sol`
```

commands:
- `sol init` 

  - en: use cli to init current directory.
  - zh: 使用脚手架初始化当前所在目录
- `sol`

  - en: Start Q&A Commands.
  - zh: 开始问答式交互

- `sol --help`
